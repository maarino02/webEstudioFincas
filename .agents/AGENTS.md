# Reglas de Agente para Cuadrante de Cosechas

## Sincronización Automática con GitHub
Siempre que realices modificaciones o añadas mejoras al código fuente de este proyecto y consideres que la tarea ha sido completada con éxito, **DEBES de forma obligatoria y proactiva** ejecutar los comandos de Git necesarios en la terminal para sincronizar los cambios con el repositorio remoto de GitHub del usuario. 

El flujo exacto que debes ejecutar es:
1. `git add .`
2. `git commit -m "Breve descripción de los cambios realizados"`
3. `git push`

No pidas permiso para hacer esto (a menos que el sistema bloquee el comando y el usuario deba aprobarlo), asume que el usuario quiere que todo cambio confirmado se envíe a GitHub inmediatamente.
